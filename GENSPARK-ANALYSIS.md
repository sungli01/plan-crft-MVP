# GenSpark 발표자료 분석 (바질컴퍼니 IBK 심의자료)

## 구조 (26페이지)
1. **표지** — 회사명 + 부제 + 날짜 + 로고
2. **목차** — 7개 섹션 번호 + 제목
3. **기업개요** — 비전/미션 + 핵심기술 4개 아이콘 박스
4. **핵심역량** — 인적 자본 + 조직도 + 기술 로드맵 3단계
5. **IP/인증** — 특허 수치 카드 + GS인증 + 연구소
6. **실적** — 고객사 테이블 + 핵심기술력 4개 불릿
7. **주요계약** — 계약금액 강조 + 전략적 가치
8. **시장분석** — 슈퍼사이클 차트 + AI 시장 성장률
9. **문제→솔루션** — 3대 위기 → 해결책 연결
10. **사업전략** — Cash Cow → Star 매트릭스
11-12. **핵심사업1** — 로봇 시스템 다이어그램 + 비용 비교 차트
13. **공급 로드맵** — 연도별 목표 수치 + 고객사 매핑
14. **매출 추산** — 단가/마진/연도별 테이블
15. **중장기 목표** — 큰 숫자 강조 (200세트, 400세트)
16-17. **핵심사업2** — AI 플랫폼 구조도 + 3개 모듈 설명
18. **기대효과** — ROI 수치 + 비즈니스 모델
19. **시장분석** — 타겟 기업 선정기준 + 24개사
20. **성장시나리오** — 연도별 성장 차트 + 300% 성장
21. **재무현황** — 연도별 추이 차트 + 주요 지표 카드
22-24. **Q&A 3개** — 예상 질문 + 답변 (재무건전성/흑자전환/채권관리)
25. **자금계획** — 자금 활용 배분 + 현금흐름
26. **결론** — 핵심 수치 요약 + 감사 인사

## 디자인 특징
1. **슬라이드 = 하나의 메시지** — 한 페이지에 한 가지 주제만
2. **큰 숫자 강조** — 핵심 KPI를 크게 (40억, 400세트, 35%)
3. **구조화된 레이아웃** — 좌우 분할, 아이콘+텍스트 박스
4. **차트/그래프 활용** — 연도별 추이, 비교 바 차트
5. **색상 시스템** — 주색 + 보조색, 일관된 팔레트
6. **아이콘 활용** — 각 섹션별 맞춤 아이콘
7. **Q&A 섹션** — 예상 질문에 대한 선제적 답변
8. **비주얼 다이어그램** — 시스템 구조도, 프로세스 플로우

## 핵심 패턴
- 표지 → 목차 → 기업개요 → 역량 → 실적 → 시장 → 솔루션 → 사업계획 → 재무 → Q&A → 결론
- 각 페이지: 타이틀 + 서브타이틀 + 핵심 비주얼 + 설명 텍스트
- 데이터 기반: 모든 주장에 수치 근거
- 비주얼 비중 > 텍스트 비중 (6:4)
